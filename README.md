# Mantus
This is a program written in C to encrypt and decrypt files recursively from the current directory.
## Prerequisite
GCC compiler
## How to run?
#### Encryption : 
```
$ gcc mantus.c
./a.out
```
#### Decryption : 
```
$ gcc mantus_decrypter.c
./a.out
```

### How to change encryption key 

edit enc.txt for changing encrytion key
